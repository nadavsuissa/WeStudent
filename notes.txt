משהו ממש חשוב - לוודא מה אתם שמים בקומיט שלכם ואיך אתם קוראים לזה, המון פעמים נכנס גם זבל. הפקודות הם:
git diff <branch name>      --- לבדוק מה פיזית כתבתם כדי להחליט מה השם של הקומיט והאם זה נכון
git status     ---   לבדוק על איזה ענף אתם ואיזה קבצים עובדו
git reset     ---  לבטל הוספה של קבצים בלי למחוק אותם
git fetch --prune --all    ---  לבדוק שינויים בלי למשוך כלום 
git checkout -b <new branch>     ---   להוסיף ענף חדש
git reset HEAD~1       ---  לבטל קומיט מבלי למחוק את הקבצים
